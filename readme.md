# DailyQuiz

## Dependencies
- Dependency Injection: koin
- API: Retrofit + Moshi
- Navigation: Compose destinations
- Database: Room

## Implemented features
1. Start screen + default quiz loading
2. Local history
3. Quiz results
4. Animations (fade in/out)
